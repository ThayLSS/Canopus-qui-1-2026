# 🎭 Diagrama de Caso de Uso

## 1. Introdução
O diagrama de caso de uso é essencial para alinhar a visão do cliente com a do desenvolvedor, definindo claramente o escopo do projeto ao descrever o comportamento do sistema sob o ponto de vista do usuário final.  

## 2. Atores Identificados
Usuário.  

## 3. Casos de Uso
Editar/Excluir registros cadastrados, Configurar alertas, Cadastrar reservas financeiras, Cadastrar receitas, Cadastrar despesas, Login do usuário, Cadastrar perfil, Baixar relatórios mensais, Gerar relatório financeiro, Realizar buscas por filtros e Visualizar registros em formato de calendário.
